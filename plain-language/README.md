# plain-language

Review text for plain language compliance based on U.S. federal [Plain Language Guidelines](https://digital.gov/guides/plain-language).

## Install

```bash
npx skills add ggwicz/skills --skill plain-language
```

## What it does

Scans documentation, READMEs, marketing copy, UI strings, code comments, and error messages for readability issues. Produces structured edit recommendations with severity ratings and before/after examples.

## Triggers

- "plain language review"
- "check readability"
- "audit the copy"
- "make this clearer"
